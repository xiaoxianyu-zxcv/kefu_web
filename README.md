# kefu

## 项目简介
这是一个基于 Vue 3 和 Vite 的客服系统，包含消息发送、接收、文件上传等功能。

## 技术栈
- Vue 3
- Vite
- Element Plus
- Pinia
- Axios
- SockJS
- STOMP.js

## 目录结构
- `src/` - 源代码目录
  - `src/App.vue` - 主应用组件
  - `src/main.js` - 应用入口文件
  - `src/router/index.js` - 路由配置
  - `src/services/` - 服务层代码
  - `src/stores/` - 状态管理
  - `src/utils/` - 工具函数
  - `src/view/` - 视图组件
- `public/` - 公共资源目录
- `README.md` - 项目说明文件
